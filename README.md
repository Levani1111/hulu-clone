# Hulu Clone 

This is my Hulu clone using the Hulu API. It allows you to pick Hulu movies and shows by genre. 

### Tech
* REACTJS
* MATERIAL UI
* FLEXBOX
* TMDB - API calls to a movie service
* React flip move for AMIMATIONS
* Delpoy on Firebase hosting
